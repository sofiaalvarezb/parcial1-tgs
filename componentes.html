Ejemplos de Componentes de Sistemas:

1Procesador (CPU):

Componente de procesamiento que ejecuta instrucciones y realiza operaciones sobre datos.

2Memoria RAM:

Almacenamiento temporal utilizado por el sistema para ejecutar programas y procesos.

3Disco Duro:

Almacenamiento de datos permanente utilizado para guardar archivos y programas.

4Pantalla:

Componente de salida que muestra información visual al usuario.

5Teclado y Ratón:

Dispositivos de entrada que permiten al usuario interactuar con el sistema.

6Sistema Operativo

Software que gestiona los recursos del sistema y proporciona una interfaz entre el hardware y el software de aplicación

7Base de Datos:

Almacenamiento organizado de datos utilizado por el sistema para almacenar información de manera estructurada.

8Red de Comunicaciones:

Sistema que permite la comunicación entre diferentes dispositivos y sistemas.

Subsitemas

Sistema Operativo:
Subsistema de gestión de memoria: Responsable de asignar y liberar memoria para programas y procesos.
Subsistema de gestión de archivos: Encargado de crear, leer, escribir y eliminar archivos en el sistema de almacenamiento.
Subsistema de administración de procesos: Controla la ejecución de programas, incluyendo la planificación de la CPU

Redes de Computadoras:
Subsistema de protocolos de comunicación: Define los formatos para la transmisión de datos a través de la red, como TCP/IP.
Subsistema de enrutamiento: Determina la mejor ruta para enviar datos de un origen a un destino a través de la red.
Subsistema de seguridad de red: Encargado de proteger la red y los datos contra amenazas externas, como detección de intrusiones.

Desarrollo de Software:
Subsistema de compilación: Convierte el código fuente en un formato ejecutable o en otro formato utilizable.
Subsistema de depuración: Facilita la corrección de errores en el código mediante herramientas como depuradores
Subsistema de control de versiones: Gestiona las distintas versiones del código fuente y facilita la colaboración entre desarrolladores.

Sistemas de Bases de Datos:
Subsistema de gestión de transacciones: Garantiza la consistencia y la integridad de los datos mediante operaciones de transacción
Subsistema de optimización de consultas: Optimiza las consultas de base de datos para mejorar el rendimiento y la eficiencia.
Subsistema de seguridad de datos: Controla el acceso a los datos y protege la integridad de la información almacenada.

Componentes de un sistema

Entradas (Inputs):

Son los datos, señales o materiales que ingresan al sistema desde su entorno externo.

Procesamiento:

Es el conjunto de operaciones o actividades que el sistema realiza sobre las entradas para producir salidas.

Salidas (Outputs)

Son los resultados o productos que el sistema genera como resultado del procesamiento de las entradas.

Almacenamiento de Datos:

Es donde se guardan los datos o la información que el sistema necesita para su funcionamiento.
